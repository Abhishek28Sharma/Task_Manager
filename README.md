## ⚙️ Installation

```bash
git clone https://github.com/Abhishek28Sharma/Task_Manager.git
cd frontend
npm install
```

### 🔑 Environment Variables

Create `.env` file:

```env
VITE_API_URL=<Backend_URL>/api/tasks
```

### ▶️ Run App

```bash
npm run dev
```
